# C - `'Variables'` | `'if'` | `'else'` | `'while'`

## 0 - (`'if'` | `'else'` | `'printf'` | `''` | )
Code: `if...else` statement
Structure:\
`if ([condition], [variable/value]){"Code to be executed eg. printf"}`

```
if (n > 0)
	{
	printf("%i is positive\n", n);
	}
	else if (n == 0)
	{
	printf("%i is zero\n", n);
	}
	else if (n < 0)
	{
	printf("%i is negative\n", n);
	}
```
